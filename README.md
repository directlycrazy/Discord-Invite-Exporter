# Discord-Invite-Exporter
 Exports invites of every server the user is in to a json file.

# Setup

* Add your user token into `config.json`
```
{
	"token": "ADD_TOKEN_HERE"
}
```

* Install the required packages using

```
npm install
```

* Run the program using

```
npm start
```
